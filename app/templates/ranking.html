API文档

所有接口暂时无需鉴权，返回 JSON。

涨跌幅排行榜

按指定时间窗口获取涨跌幅最高的合约列表。

GET /api/rankings/change?window=24h&limit=50&direction=long

交易量排行榜

按指定时间窗口获取计价货币成交量最高的合约列表，例如 USDT 成交量。

GET /api/rankings/volume?window=24h&limit=50&direction=short

K线数据

查询指定合约的本地 1H K 线数据。

GET /api/candles?inst_id=BTC-USDT-SWAP&limit=100

请求参数

参数 类型 必填 默认值 说明
window string 否 24h 排行时间窗口。可选：1h、2h、4h、12h、24h。
limit integer 否 50 返回排行条数。取值范围：1 到 500。
direction string 否 无 按多空方向过滤。可选：long、short。不传时返回全部方向。

返回字段

字段 类型 说明
metric string 排行指标。pct_change 表示涨跌幅，volume 表示交易量。
window string 本次查询的时间窗口。
limit integer 本次请求的返回条数上限。
collector_mode string 当前 K 线采集模式，rest 或 websocket。
data array 排行榜数据列表。
rank integer 排名，从 1 开始。
inst_id string OKX 合约 ID，例如 BTC-USDT-SWAP。
direction string 多空方向。long 表示多，short 表示空。按涨跌幅判断：大于等于 0 为多，小于 0 为空。
pct_change number 涨跌幅百分比。例如 2.31 表示上涨 2.31%。
abs_pct_change number 绝对涨跌幅，用于涨跌幅榜默认排序。
volume_quote number 计价货币成交量，默认为 USDT 口径。
open_price number 时间窗口起始 K 线的开盘价。
close_price number 时间窗口末尾 K 线的收盘价。
start_ts integer 计算窗口起始时间戳，OKX K 线毫秒时间戳。
end_ts integer 计算窗口结束时间戳，OKX K 线毫秒时间戳。
calculated_at integer 排行榜计算时间，Unix 秒时间戳。
